# Dark Wintertodt

Dark Wintertodt is a RuneLite plugin that reduces the brightness of the Wintertodt arena.

It darkens the main Wintertodt play area, including the raised floor sections and surrounding structures, so the area is easier on the eyes.

## What It Does

- Darkens the Wintertodt arena while you are inside the Wintertodt regions
- Applies to the main floor, raised floor sections, walls, and ceiling/roof structures used in the arena
- Turns off automatically when you leave the Wintertodt area

## Settings

- `Darkness Strength`: Controls how strong the darkening effect is

## Notes

- This plugin is only intended for Wintertodt
- The effect is visual only
- This plugin is not compatible with 117 HD 
- There may be flashes when moving around but those should stabilize once you are in the game area. (caused by reloading new tiles)
